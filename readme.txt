bestsvm.dat是SVM训练的结果。以此判断音符的种类。
coeff.npz是摄像头矫正的数据。不同的摄像头矫正数据不同。
globallist.py含控制输出的全局变量。
main.py是主程序。
picture.py是图像处理所有函数。
音乐播放的函数放在main.py里了。
line.txt是五线谱五条线的信息。

2018.3.3：
目前遇到的问题有：音符识别错误。就是说机器学习错误。 原因可能是用来训练的数据是未经摄像头矫正的音符，应用时的音符是经过矫正的。也有可能是样本图片的像素太少，导致二分音符和四分音符分辨不出来。
五线谱识别不出来。一种办法是打强光。一种办法是手动标注五线谱的位置。一种办法是用固定位置。把乐谱板跟架子放在一起。
